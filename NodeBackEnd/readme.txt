#sophyai.space back end.

The flow file  is a prototype of back end nodered application  support sophyai.space network , for install you need a simple nodered application and import the flow. 
It's a prototype of network available for external contribution is great.
It manage the ground station control protocol by a mqtt server like mosquitto.
the telegram and discord bot.
the interface with influxdb.
And lot of other functionality that will be available in future release.
For more informatio ask support in our discord community in #backend-dashboard-dev channel.
